## User Management
### Add User
```Shell
sudo adduser BENUTZER [OPTIONEN]
sudo addgroup [OPTIONEN] GRUPPE
```
Examples
```Shell
sudo adduser newuser      # create new user
sudo adduser xyz www-data # add user to group
```
Add given user to sudo group
```Shell
sudo usermod -aG sudo BENUTZERNAME
```
